Comandos git de básico a intermedio

Git es un sistema de control de versiones que nos ayuda a gestionar y almacenar los cambiós de nuestros proyectos. Pensando eso tiene tanto poder que puede llegar a convertirse en una red social de programadores en la cual se comparten miles de lineas de código, de esa manera se motiva el codigo open source y el trabajo colaborativo.
